# Qwen Agents Toolpack Extension

## Overview
You have access to specialized agents and tools from the converted Claude plugins. These provide domain-specific expertise across multiple areas.

## Available Tools
When appropriate, you can use these specialized tools to leverage expert knowledge:

- `git-advanced-workflows` - Expert in advanced Git workflows (rebasing, cherry-picking, bisect, worktrees, reflog)
- `backend-architect` - Expert in backend architecture, API design, and microservices
- `python-pro` - Expert in modern Python development with best practices
- `security-auditor` - Expert in security auditing and vulnerability assessment
- `frontend-developer` - Expert in frontend UI development
- `database-architect` - Specialist in database architecture and optimization
- `cloud-architect` - Expert in cloud architecture for AWS/Azure/GCP

## Usage
When a user requests expertise in any of these domains, use the appropriate specialized tool to provide more accurate and detailed assistance. This extension provides access to knowledge from over 65+ converted Claude plugins with 152+ agents and 61+ tools.

Apply the right tool based on the user's request to provide expert-level guidance in that specific domain.
